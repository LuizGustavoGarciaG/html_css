FAQ - Perguntas Frequentes

O que é HTML?

HTML é a linguagem de marcação usada para estruturar páginas web.

O que é CSS?

CSS define o estilo visual de páginas HTML, como cores, fontes e espaçamento.

O que é JavaScript?

É uma linguagem de programação usada para adicionar interatividade a sites.

Como hospedar um site?

Você pode usar serviços como GitHub Pages, Netlify ou Vercel para hospedar gratuitamente.

O que é um domínio?

É o endereço do seu site na web (exemplo: www.meusite.com).
